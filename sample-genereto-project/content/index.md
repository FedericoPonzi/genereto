# Welcome!

This is a sample website built with [Genereto](https://github.com/FedericoPonzi/genereto).

## Features Demonstrated

- **Markdown content** with YAML frontmatter
- **Template includes** for reusable components
- **Blog support** with RSS feed generation
- **SEO metadata** (title, description, keywords, Open Graph)

## Getting Started

1. Edit this file (`content/index.md`) to customize your home page
2. Add blog posts in `content/blog/`
3. Customize templates in `templates/main/`
4. Run `genereto --project-path .` to build

Check out the [blog](blog/index.html) for more examples!
