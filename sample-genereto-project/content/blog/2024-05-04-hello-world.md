Welcome to your first blog post! This demonstrates basic Genereto features.

## Writing Content

Write your content in **Markdown** format. Genereto converts it to HTML automatically.

### Supported Features

- **Bold** and *italic* text
- [Links](https://github.com/FedericoPonzi/genereto)
- Lists (like this one!)
- Code blocks
- Blockquotes

> This is a blockquote. Great for highlighting important information.

## Code Blocks

Add code with syntax highlighting:

```rust
fn main() {
    println!("Hello, Genereto!");
}
```

## What's Next?

1. Edit this post to add your own content
2. Create new posts by adding `.md` files to `content/blog/`
3. Customize the templates in `templates/main/`

Happy blogging!
